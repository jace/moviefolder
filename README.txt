MovieFolder: Get ratings and folder cover images from IMDb

Instructions:

1. sudo easy_install imdbpy

2. cd ~/Movies

3. ls -1F
   Movie Name 1 (2010)/
   Movie Name 2 (1957)/
   Movie Name 3 (2011)/

4. ~/path/to/download/folder/movie-info.py

5. ls "~/Movies/By Rating"; ls "~/Movies/By Genre"

Enjoy!
